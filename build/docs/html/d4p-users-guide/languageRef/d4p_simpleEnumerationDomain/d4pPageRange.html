d4pPageRange

d4pPageRange holds a page range value. It is primarily intended for capturing page numbers during legacy conversion or for identifying literal page number references in legacy content. Processors can then do something different for non-print outputs, such as providing a link to the actual place in the output for the content that got referenced by the page numbers. For digital publications, the page numbers should be omitted in most cases.
d4pPageRange is a specialization of data.
