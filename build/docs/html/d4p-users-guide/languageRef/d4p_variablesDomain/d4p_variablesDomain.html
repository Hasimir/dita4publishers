d4p_variablesDomain

d4p_variablesDomain defines elements for defining "variables" and variable references that are metadata based. This domain may be used in maps and topics as all of the elements defined in it are sensible in both contexts. WARNING: This domain is entirely experimental. There is no guarantee for ongoing support for this domain. The markup details may change without warning. Due to the experimental nature of this domain, no further documentation is provided at this time; when the domain is stable, documentation will be provided. Interested users should see the domain definition files.
