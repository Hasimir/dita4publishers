d4p_simpleBiblioentry

The d4p_simpleBiblioentry element is the element for a simple bibliographic entry. It is analogous to DocBook's biblioentry and bibliomixed elements.
This element specializes from the p element. To provide further markup within the element use ph with an outputclass, use d4p_simpleBiblioset with an outputclass, or further specialize d4p_simpleBiblioset, in which case you might consider such element names as d4p_simpleBiblioAuthor, d4p_simpleBiblioTitle, etc.
