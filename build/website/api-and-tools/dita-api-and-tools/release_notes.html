Release 0.1 2010-01-04

This is the first release of this package. See the main documentation for package details.

Known Bugs

No known bugs.

Known Issues

  • Support for DITAVAL files not yet implemented, so the -d option for the map reporter and DXP packager have no effect.
  • DXP packager does not yet support creation of DXP manifest maps (partly because the map design hasn't been started, much less completed).
  • The where-used features of the DITA Link Management Service are not yet implemented.
