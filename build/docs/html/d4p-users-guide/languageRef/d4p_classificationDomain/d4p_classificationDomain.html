d4p_classificationDomain

The d4p_classificationDomain defines specializations of data for classifying components of publications (topics, figures, tables, etc.). This clearly identifies the metadata elements it contains as being classifying as opposed to any other type of metadata (such as identifying metadata). Classifying metadata normally relates its container to items in defined classification taxonomies.
