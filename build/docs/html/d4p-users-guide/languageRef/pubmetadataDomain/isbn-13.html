isbn-13

isbn-13 represents the 13-digit International Standard Book Number of a publication. If the publication has a 10 digit ISBN or an ISBN in a different format, then the appropriate isbn-10 or isbn element should be used. It is left to the user on whether to separate parts of the ISBN with hyphens or other separators, but consistency should be valued.
isbn-13 is a specialization of data.
