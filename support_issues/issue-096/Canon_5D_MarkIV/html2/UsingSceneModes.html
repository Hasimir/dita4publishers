Using Scene Modes

With scene modes, the camera still performs most functions automatically, but you exercise more control by describing the shooting environment to the camera. Examples of scene modes include night portrait, beach and snow, and sports.

To use one of the scene modes:

  1. Rotate the Mode dial to SCN

  2. Press the Menu button to view the first scene mode.

  3. Rotate the main dial to scan through all the scene modes. Keep rotating until you find the one you want.1

  4. Press the Set button to select the displayed scene mode.

  5. Use the camera like a point and shoot camera.
