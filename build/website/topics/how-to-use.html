How To Use the DITA For Publishers Materials

You can use the DITA For Publishers materials in the following ways:
  • Use the Toolkit plugins to process publication maps (or maps of any type)
  • Use the sample data as examples of how to represent different types of publications using the Publishing specializations
  • Use the specialization modules as the base for your own Publishing-related specializations
  • Use the Word-to-DITA transformation framework to convert consistently-styled Word documents into DITA maps and topics.
  • Use the DITA-to-InDesign transforms to generate InCopy and InDesign documents from DITA source.

As provided, the plugins and documents can be used immediately with the DITA Open Toolkit. The Toolkit plugins have been tested with the 1.5 Open Toolkit. They may work with the 1.4.3 Toolkit but that use is not supported.

You should be able to use the materials with any DITA-aware XML editor that uses the Open Toolkit to resolve DTD references, such as OxygenXML. All the materials have been developed and tested using OxygenXML primarily, but they do not have any Oxygen-specific aspects or data (nor anything specific to any other editor).

Note that currently there are only DTD versions of the vocabulary modules, so tools that require XSD schemas will not be able to use them right now. XSD versions can be generated almost automatically using the trang DTD-to-XSD conversion tool (integrated with the OxygenXML editor). These XSD files will work but will not themselves be conforming DITA vocabulary modules. If you have a need for XSD versions of any vocabulary modules, please post that requirement to the Yahoo DITA User's mailing list.

See the DITA for Publishers User Guide for complete and most-up-to-date information on using the DITA for Publishers materials.
