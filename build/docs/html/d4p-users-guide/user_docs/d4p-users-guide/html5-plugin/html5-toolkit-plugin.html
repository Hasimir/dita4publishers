Chapter 6. HTML5 Plugin

Generates HTML5 from DITA maps.

The HTML5 transformation type produces HTML5-based output that integrates HTML5 technologies, including AJAX and CSS. The resulting HTML5 provides dynamic and modern navigation and presentation. The supporting components such as the CSS style sheets and JavaScript used to render the HTML in browsers is designed to be easily adapted or replaced as required to provide specific navigation and presentation styles.

Note that HTML5 does not use the HTML frames mechanism. All navigation and visual organization of content within the browser is done through JavaScript.

  5. Notes

    Notes are output with the aside element except when @importance is set to low or obsolete, which will output a details elements. Details elements should be collapsed by default on new browsers.
