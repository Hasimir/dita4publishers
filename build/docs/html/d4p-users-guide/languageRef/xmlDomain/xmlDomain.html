xmlDomain

The xmlDomain provides inline elements for identifying mentions of XML constructs: element types, attributes, text entities, parameter entities, and numeric character references. It is convenient for documents that discuss XML markup (such as D4P User Guide). NOTE: This domain will be replaced by its equivalent with DITA 1.3.
