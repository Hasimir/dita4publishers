DITA For Publishers Java API and Tools

The DITA for Publishers (D4P) Java API and tools provide a general-purpose API for DITA link management, as well as various utilities for working with collections of DITA documents. These APIs are intended to be a public and open API suitable for use by a variety of tools and applications, including DITA-aware editors, content management systems, and processors. The tools are intended to serve as both tests and demonstrations of the APIs as well as useful utilities.

The D4P API (dita4publishers-dita-api.jar) provides the following general services and classes:
  • DITA Link Management service, which provides general facilities for working with DITA-defined links and addresses, especially the management of DITA key spaces and resolution of key references.
  • DITA key space classes and APIs
  • Conditional processing (filtering) services
  • DITA "bounded object set" (BOS) management. A DITA bounded object set is the set of resources (DITA documents and non-DITA resources) used in the context maps and topics ultimately used from a given root map.

See the Javadocs for the API and implementation classes for details. These topics provide a general conceptual and design overview for the API and tools.

  • DITA Link Management API
    The DITA link management API provides a set of facilities for working with and managing links within DITA documents, in particular keys and key references. The API is intended to be useful and usable by a variety of different tools types: editors, content management systems, processors, and so on. The API enables answers to two basic questions: "what resource is a given key bound to?" and "Where is a given potential link target used within a given set of DITA documents?".
  • DITA For Publishers Tools
    The DITA for Publishers tools provide implementations of the DITA Link Management API as well as utilities for using the implementations and doing useful things, such as getting key space reports and packaging DITA bounded object sets as Zip files (DITA Interchange Packages (DXP) packages).
