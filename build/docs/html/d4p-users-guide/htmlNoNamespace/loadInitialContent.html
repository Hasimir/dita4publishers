d4p.loadInitialContent

Control if the initial content should be loaded if no content is specified in the url.

d4p . loadInitialContent = boolean (true or false)

Description

Control if the first link in the navigation should be loaded by d4p once the page load or not. Most situations will require this property to be set to true.

In the situation where you need a custom first page based on DITA semantic (meta, authors or whatever), you would be abble to diseable this javascript feature by setting up the value to false.
