Running the html5 plugin

The html5 plugin simply add the html5 transtype to the DITA-OT

In order to run the plugin, you must be familiar with the process of running the DITA-OT. Please refer to the documentation.

Running DITA-OT from Ant

Execute the usual command using html5 as the transtype

ant -Dargs.input=samples/hierarchy.ditamap 
    -Doutput.dir=out/garage/html5 
    -Dtranstype=html5

Running DITA-OT from command-line tool

java -jar lib/dost.jar 
    /i:samples/hierarchy.ditamap 
    /outdir:out/garage/html5 
    /transtype:html5
