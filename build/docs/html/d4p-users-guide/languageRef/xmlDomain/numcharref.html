numcharref

A numcharref is an inline element for numeric character references, which are short sequences of characters that start with & and end with a ; and represents a single character. It can only contain other elements of the keyword and text types.
This is a specialization of keyword. The markup <numcharref>2014</numcharref> (em-dash) should produce &#2014; (em-dash)

Inheritance

+ topic/keyword xml-d/numcharref 

To use an em-dash in your output, use this in your XML: <numcharref>2014</numcharref>.
