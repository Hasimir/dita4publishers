pubmetadataDomain

The pubmetadataDomain contains a rich set of elements to be used for representing metadata information for publications that meets the needs of publishers (such as that which describes, names, and classifies the publication, including its title, any ISBN or ISSN numbers, copyright statements, authorship, ownership, and so on). Publishing publication metadata tends to be more demanding than that required by technical documents. As a map domain, the publishing metadata elements can be added to any map type. To use this, integrate the domain into your local map types.
