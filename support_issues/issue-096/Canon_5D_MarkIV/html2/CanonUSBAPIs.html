Canon USB APIs

Canon provides an application programming interface (API) to enable software developers to access images stored on the SD card to and to control camera functions.

Introduction

The Canon USB APIs enable you to perform the following functions with 3rd party software when the camera is attached to a computer with a USB inteface.

  • Fetch a range of thumbnail images

  • Fetch an individual image

  • Control camera functions for remote shooting

The APIs are especially useful for enabling tethered shooting. The following sections document the various APIs.

USBFetchImage

Fetch an image by name or by sequence number on the SD card.

JPEGImage USBFetchImage (
	string imagename;
	integer imagesequence;
);
Argument Description
imagename A string giving the full image name. If not null, indicates that you are requesting a named image. If null, indicates you are requesting an image by sequence.
imagesequence An integer indicating the position on the SD card of the image to retrieve. If zero, retrieves the next image on the card.

Retrieving Images in Sequence

The example demonstrates how to retrieve images in sequence starting from the first image on the card.

JPEGImage myJPEGImage; 
String noname = "";    // Must pass in a null image name if requesting next image in sequence
myJPEGImage = USBFetchImage(noname, 0);
// do something with the image

Retrieving a Named Image

This example demonstrates how to retrieve an image by name.

JPEGImage myJPEGImage; 
String imgName = "img_2003.jpg";    // imagesequence argument is ignored if this is not null.
myJPEGImage = USBFetchImage(imgName, 0);
// do something with the image
