xmlatt

An xmlelem is an inline element for identifying an attribute of an XML element.
This is a specialization of keyword. The markup <xmlatt>outputclass</xmlatt> should produce the output: @outputclass.

Inheritance

+ topic/keyword xml-d/xmlatt 

You can use <xmlatt>outputclass</xmlatt> on many elements to distinguish them in an output, such as EPUB or PDF.
