Directory structure

The following images shows the structure of the plugin directory.

libs

The libs directory contains a collection of either javascripts or css librairies that might be usefull for projects. Popular libraries includes:
  • jQuery
  • jQuery UI
  • 960 css grid
  • jQuery BBQ widget

The library also contains the html5 library that has been developped for this project.

The lib directory contains an ant project (build.xml) which allows to create custom builds from the library to the themes directory.

old

Deprecated

samples

Contains samples code

sites

This directory contains configuration to build custom sites. For now, sites contains only one directory, name default, which contains all information to process certain libraries to build the default theme.

themes

The theme directory contains themes used for production. You should never edit content in this directory.

The content of this directory is build using the sites intructions using the ant build.xml process found in the lib directory.

xml2json

This directory contains the xml2json library developped by Bram Stein. There is no license associated with this project.

xsl

The xsl directory contains all xsl for this project.
