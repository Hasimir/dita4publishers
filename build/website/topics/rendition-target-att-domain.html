Rendition Target Attribute Domain

Provides a @props specialization for filtering and flagging on rendition target (e.g., PDF, HTML, EPUB, etc.)

The rendition target attribute domain provides the @props specialization attribute @d4p_renditionTarget. The domain does not define the allowed values for the attribute (since the set of interesting rendition targets is unbounded) but typical values would be "pdf", "html", 'epub", "kindle", etc. This attribute allows you to have map or topic elements that are specific to a particular rendition target.
