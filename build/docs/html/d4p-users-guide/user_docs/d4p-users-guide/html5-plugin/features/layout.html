Layout

The HTML5 plugin offers 2 layouts:

  • left aligned;
  • centered layout, the page should have a drop shadow to mimics a page.

In both cases, the navigation appears on the continuous block on the left of the page.

The layout is control by the 960 Grid System. The size of the navigation block and the main content is fully adjustable via the ant properties. The information is described in the developper section.

All xsl templates have been splitted in a way that developper should be abble to create their own layout easely.
