isbn-10

isbn-10 represents the 10-digit International Standard Book Number of a publication. If the publication has a 13 digit ISBN or an ISBN in a different format, then the appropriate isbn-13 or isbn element should be used. It is left to the user on whether to separate parts of the ISBN with hyphens or other separators, but consistency should be valued.
isbn-10 is a specialization of data.
