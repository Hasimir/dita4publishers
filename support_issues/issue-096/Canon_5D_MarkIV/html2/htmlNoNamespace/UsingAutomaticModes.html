Chapter 4. Using Automatic Modes

The camera has two types of automatic modes: Intelligent Auto and scene modes.

Intelligent Auto makes all the decisions for you. With scene modes, you exercise a bit more control by telling the camera about the environment in which you are shooting.

  • Using Intelligent Auto Mode
    Intelligent Auto mode makes all the shooting decisions for you.
  • Using Scene Modes
    With scene modes, the camera still performs most functions automatically, but you exercise more control by describing the shooting environment to the camera. Examples of scene modes include night portrait, beach and snow, and sports.
  • Scene Modes Descriptions
  • Creative Auto Shooting
    The Creative Auto (CA) mode enables you to exercise more control over the decisions the camera makes when shooting in Intelligent Auto.
