The d4p properties

  • d4p.version
    Version of the current d4p library
  • d4p.loadInitialContent
    Control if the initial content should be loaded if no content is specified in the url.
  • d4p.hash
    Contains information of the current hash
  • d4p.outputSelector
    jQuery element selector in which the content should be output
  • d4p.navigationSelector:
    jQuery selector of the element containing the navigation.
  • d4p.externalContentElement
    Once a AJAX call is performed, the d4p.externalContentElement is used to defined which element of the response hold the content to be displayed
