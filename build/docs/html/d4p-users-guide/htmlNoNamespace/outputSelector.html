d4p.outputSelector

jQuery element selector in which the content should be output

d4p . outputSelector = string (default: "#d4h5-main-content")

Description

Allows to specify the element selector in which the main content will be displayed.

Note: Be aware that an ant property exists also to control the id of the element in the xsl. Please change value at both place if you need to change them.
