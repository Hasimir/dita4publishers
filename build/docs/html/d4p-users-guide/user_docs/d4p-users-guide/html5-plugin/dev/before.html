Before starting a new theme

Before starting to develop a theme or adapt your to fit in this plugins, you need to install the yui compressor on your computer.

In order to be abble to compress all javascripts and css files

  1. Download the YUI compressor
  2. Move the directory into the lib directory of the html5 plugin so the final path of the jar file is: DITA-OT/plugins/net.sourceforge.dita4publishers.html5/libs/yuicompressor-2.4.7/build/yuicompressor-2.4.7.jar

    The html5 plugin has been developped with the yuicompressor version 2.4.7. If you use a different version, you might have to edit the file shared.properties under the lib directory, which tells ant where to find the YUI compressor.

  3. Run a test on your shell: 
    shell: cd DITA-OT
shell: ./startcmd.sh
shell: cd plugins/net.sourceforge.dita4publishers.html5/libs
shell: ant -f build.xml

    This steps takes independantly all the css and js defined in the sites/default/default.properties, concatenate them into one style.css and one script.js respectively and compress them using the YUI compressor to style-min.css and script-min.js respectively in the themes/default directory.

    You should see no error in the package.compress execution target.
