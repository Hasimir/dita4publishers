This is the new home for the DITA for Publishers project (migrated from it's original home on SourceForge).

This code base represents version 1.0.0 of the DITA for Publishers project, reflecting a name change in all the plugins from net.sourceforge.dita4publishers.* to org.dita4publishers.*.

The source code has been reorganized into separate git repositories, one for each Open Toolkit plugin, one for the sample data, and one for the RSuite CMS-specific stuff.
