d4pPageNumber

d4pPageNumber holds a page number value. It is primarily intended for capturing page numbers during legacy conversion or for identifying literal page number references in legacy content. Processors can then do something different for non-print outputs, such as providing a link to the actual place in the output for the content that got referenced by the page number. For digital publications, the page number should be omitted in most cases.
d4pPageNumber is a specialization of data.
